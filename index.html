iPinYou Real-Time Bidding Dataset
for Computational Advertising Research

Data Dump [zip file] (5.87G)

The data server is in University College London, UK.

What is this dataset?

This dataset contains the data of three seasons of iPinYou global RTB bidding algorithm competition[link].
